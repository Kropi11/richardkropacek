## task02
Bitové posuny:
 - 1<<0 = 0b1
 - 1<<1 = 0b10
 - 1<<i = 0b100... (i krát 0)
 - 60>>4: 60 = 0b101000; 60>>4 = 0b10 (poslední 4 bity zahodíme)

`byte x`: horní 4 bity: `x>>4`, dolní 4 bity `x & 0xF`
